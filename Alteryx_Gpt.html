ChatGPT Column Classification

This custom Alteryx plugin uses OpenAI's GPT to classify the values in a given input column. The user can specify the prompt to use and choose the format and category to classify the input as. The plugin can output up to three columns with the classification results.

To use this plugin, connect the input column to the custom tool and configure the classification settings using the UI. Then, run the workflow to classify the input column and generate the output.

Requirements:

  • Alteryx Designer with Python SDK installed
  • Python packages: openai, pandas

Authentication

The plugin uses an OpenAI API key for authentication. Please enter your API key in the configuration panel before using the plugin.
